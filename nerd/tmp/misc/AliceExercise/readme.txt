To start the evolution application use one of the start scripts.

Example:

Windows: sim.bat

Linux / Mac: ./sim.sh

The binaries of the ORCS application have to be in the parent directory of this directory. For exercise 8 you need only application OrcsEvaluation.

Sample networks are given in folder networks. Please change each network and leave your solution with the given name in folder networks.
